write

  • write(data: string, callback?: function): void
    • Defined in node_modules/node-pty/typings/node-pty.d.ts:75

    Writes data to the socket.

    Optional parameter callback will be called with true if the entire data was flushed successfully to the kernel buffer or called with false if all or part of the data was queued in user memory. 'drain' event will be emitted when the buffer is again free.

    Parameters

    • data: string

      The data to write. If you want to simulate pressing the ENTER key, like when entering a command, finish the string with the character '\r'

    • Optional callback: function

      Function called with true or false as explained in the method description. Take in consideration that this listener could be called several times.

        • (flushed: boolean): any

        • Parameters

          • flushed: boolean

          Returns any
